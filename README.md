## 项目背景

本项目是**飞书AI大赛企业命题赛道**参赛作品，命题企业为**全棉时代**。

全棉时代作为母婴棉品行业的领军品牌，面临着新品研发周期长、市场验证成本高、消费者需求快速变化等挑战。如何在新品正式上市之前，准确预测市场反应，降低新品失败风险，是企业最核心的业务问题。

---

## 企业命题分析

### 命题解读

全棉时代的核心诉求：**如何利用AI技术提升新品研发效率，降低市场风险，实现产品创新的快速迭代**。

### 评审维度

- **对企业业务问题的理解**：是否准确把握母婴棉品行业的核心痛点
- **AI解决方案设计能力**：方案的完整性、创新性和技术可行性
- **技术架构完整性**：系统设计的合理性、可扩展性和落地性
- **创新性和落地价值**：是否具有独特的创新点和明确的商业价值

---

## 核心痛点

### 行业痛点

1. **新品上市风险高**：传统市场调研周期长、成本高，且难以准确预测消费者真实购买意愿

2. **消费者需求变化快**：母婴消费群体偏好变化迅速，传统调研方法滞后于市场变化

3. **用户反馈收集难**：海量用户评论分散在各个平台，人工分析效率低下

4. **产品-市场匹配困难**：产品团队难以精准把握消费者对新品设计的真实反馈

### 企业痛点

> **"这个产品设计出来，消费者到底会不会买？"**

这是全棉时代产品团队在新品研发过程中最常问的问题，也是最核心的业务痛点。

## 项目介绍

### 项目名称

**CottonMind —— AI新品共创实验室**

### 项目定位

基于飞书AI生态的虚拟消费者测试与产品创新智能体系统。

### 核心理念

帮助消费品牌在新品正式上市之前，通过AI模拟目标消费者进行虚拟市场验证。

### 核心流程

```
用户反馈 + 行业数据 + 品牌知识
        ↓
  Insight Agent 生成消费者洞察
        ↓
  构建虚拟消费者模型（画像 + 决策逻辑）
        ↓
  产品团队输入新品概念
        ↓
  AI模拟不同消费者群体反馈
        ↓
  Product Agent 优化产品方案
        ↓
  Marketing Agent 生成营销内容
        ↓
  Growth Agent 提供迭代建议
```

---

## 核心创新

### AI虚拟消费者模拟

系统不是简单分析用户评论，而是基于真实数据构建具有**画像**和**决策逻辑**的虚拟消费者，模拟真实市场反馈。

#### 创新点

1. **虚拟消费者建模**：从真实用户数据中提取消费者画像特征和购买决策逻辑

2. **动态反馈模拟**：针对不同产品设计方案，模拟不同消费者群体的差异化反馈

3. **迭代优化闭环**：根据虚拟测试结果，自动优化产品方案和营销策略

4. **飞书生态深度融合**：将虚拟消费者测试能力无缝嵌入企业日常工作流

## Agent架构

### 1. Insight Agent（核心）

**定位**：AI消费者研究专家 + 虚拟消费者生成器

**功能**：
- 分析用户评论，提炼消费痛点
- 生成消费者画像（年龄、地域、消费偏好、价格敏感度等）
- 构建虚拟消费者决策模型
- 模拟消费者测试反馈

**输入**：品牌信息、产品品类、用户评论数据、行业报告

**输出**：消费者洞察报告、虚拟消费者模型、测试反馈结果

### 2. Product Agent

**定位**：AI产品经理

**功能**：
- 根据消费者反馈优化产品方案
- 输出产品定位和核心卖点
- 提出产品迭代建议

**输入**：新品概念、虚拟消费者测试反馈、消费者洞察

**输出**：优化后的产品方案、产品定位报告、核心卖点清单

### 3. Marketing Agent

**定位**：AI营销专家

**功能**：
- 根据产品方案生成营销内容
- 输出小红书文案、短视频脚本、电商详情页

**输入**：产品方案、核心卖点、目标人群

**输出**：多平台营销内容、传播策略

### 4. Growth Agent

**定位**：AI增长分析师

**功能**：
- 分析市场反馈数据
- 提供迭代优化建议
- 预测市场表现

**输入**：虚拟测试结果、营销效果数据、行业趋势

**输出**：增长分析报告、迭代建议、市场预测

## 技术架构

### 飞书AI生态

| 飞书能力 | 应用场景 |
|---------|---------|
| 飞书群聊/智能入口 | 用户交互界面，产品团队发起虚拟测试 |

- **Multi-Agent协作**：多个Agent按业务流程协作完成任务
- **虚拟消费者建模**：从真实数据中构建消费者决策模型

### 早期验证方案

- **Dify Workflow**：用于早期Agent逻辑验证和流程调试

## 快速开始

### 环境要求

- Python 3.10+
- Dify平台账号（用于早期验证）
- 飞书企业账号（用于生产部署）

### 安装步骤

```bash
# 克隆项目
git clone <repository-url>
cd cottonmind-ai

# 安装依赖
pip install -r requirements.txt

# 启动演示
streamlit run src/demo/app.py
```

### Dify工作流导入

1. 登录Dify平台
2. 导入 `dify/workflows/cottonmind_workflow.json`
3. 配置知识库连接
4. 测试Agent协作流程
