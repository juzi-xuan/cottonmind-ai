## 一、项目亮点

### 1.1 核心创新：三维度虚拟测试体系

**创新点**：PetMind的核心创新在于**三维度虚拟测试**——从宠物适口性、功效验证和宠主购买决策三个维度进行虚拟市场验证，解决宠物健康零食研发的核心痛点。

**解决的核心问题**：

- "这个产品做出来，宠物愿不愿吃？" —— 适口性预测
- "功效能否快速验证？" —— 营养成分分析、钙磷比评估、功能性预测
- "主人愿不愿买？" —— 虚拟宠主测试

### 1.2 技术创新

- **AI虚拟适口性预测**：基于历史配方数据和用户反馈，建立适口性预测模型，在真实宠物测试前完成低成本筛选
- **AI功效验证加速**：基于宠物营养学知识，评估钙磷比、蛋白质含量等关键指标，预测功能性成分效果（美毛、护肠、洁齿）
- **三维宠主建模**：画像特征 + 决策规则 + 情感表达，构建逼真的虚拟宠主（关注成分透明度、钙磷比、功能性）
- **数据驱动权重学习**：基于评论情感分析自动提取决策权重
- **可验证的模拟结果**：每个虚拟宠主的反馈都有明确的决策逻辑支撑

### 1.3 飞书生态深度融合

- 飞书aily自定义智能体部署四个核心Agent
- 飞书多维表格管理测试数据、配方数据、功效验证结果
- 飞书工作流编排自动化测试流程（适口性→功效→虚拟宠主）
- 飞书机器人推送测试结果通知

### 2.2 核心流程

```
用户反馈 + 配方数据 + 宠物营养知识
        ↓
  Insight Agent 生成消费者洞察
        ↓
  构建虚拟宠主模型（画像 + 决策逻辑）
        ↓
  产品团队输入新品概念（含配方信息）
        ↓
  AI宠物适口性预测 → 功效验证分析 → 虚拟宠主测试反馈
        ↓
  Product Agent 优化产品方案（钙磷比调整、功能性增强）
        ↓
  Marketing Agent 生成营销内容（突出功能性卖点）
        ↓
  Growth Agent 提供迭代建议（功能性零食市场机会）
```

### 2.3 AI虚拟适口性预测

**输入**：原料组合、营养成分（钙磷比等）、产品定位、历史反馈

**输出**：适口性风险预测、可能问题、优化建议

**核心价值**：在进入真实宠物测试前完成低成本筛选，提高研发效率。AI不是替代真实宠物实验。

### 2.4 AI功效验证加速

**输入**：配方信息（原料组合、营养成分、钙磷比）、功能性成分、产品定位

**输出**：营养成分评估、功效预测（美毛/护肠/洁齿）、风险预警

**核心价值**：将传统1.5-2年的功效验证周期缩短至数小时，在进入活体试验前完成功效评估和风险预警。

### 2.5 虚拟宠主构建方法论

**七个阶段**：

1. **数据准备**：采集电商评论、小红书笔记、行业报告、配方数据、宠物营养知识
2. **消费者聚类**：基于语义相似度的K-Means聚类，识别5-10个细分群体（如Z世代成分党、科学养宠家长等）
3. **画像特征提取**：LLM提取年龄、地域、消费能力、养宠情况等特征
4. **决策逻辑建模**：基于情感分析提取决策规则和权重（如成分透明度 > 功能性 > 钙磷比 > 品牌 > 价格）
5. **功效验证模型构建**：基于宠物营养学知识，构建钙磷比评估和功效预测模型
6. **虚拟宠主生成**：整合画像+决策逻辑+功效知识，生成完整模型
7. **模拟测试**：输入新品概念，执行适口性预测→功效验证→决策逻辑，生成反馈评分

### 2.6 关键算法

**权重学习公式**：

```
W_i = (P_positive_i - P_negative_i) × F_i
```

其中：

- W\_i：特征i的权重
- P\_positive\_i：正面情感概率
- P\_negative\_i：负面情感概率
- F\_i：提及频率

**评分计算**：

```python
def calculate_purchase_intent(product_concept, decision_rules):
    total_score = sum(rule['weight'] * match_score for rule in decision_rules)
    return normalize(total_score, 1, 10)
```

## 三、落地价值

### 3.1 商业价值

| 指标      | 说明                          |
| ------- | --------------------------- |
| 新品成功率提升 | 通过三维度虚拟测试（适口性+功效+宠主）筛选高潜力产品 |

### 3.2 企业价值

1. **降低新品上市风险**：提前验证产品市场接受度、宠物适口性和功效可行性

3. **提升产品决策效率**：数据驱动的产品优化决策（钙磷比调整、功能性增强）

## 九、风险与应对

### 9.1 技术风险

| 风险          | 应对措施                 |
| ----------- | -------------------- |
| 虚拟宠主预测准确率不足 | 基于真实数据持续迭代优化模型       |
| 适口性预测准确率不足  | 积累更多配方数据和测试反馈，优化预测模型 |

| 大模型推理成本高    | 优化Prompt、使用轻量级模型     |

### 9.2 业务风险

| 风险       | 应对措施             |
| -------- | ---------------- |
| 企业数据接入困难 | 提供多种数据导入方式       |
| 用户接受度不足  | 提供直观的可视化界面和易用的交互 |
| 隐私合规问题   | 数据私有化部署、脱敏处理     |
