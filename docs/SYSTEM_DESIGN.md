# CottonMind System Design

## Agent Architecture

### 1. Insight Agent

**身份：**
消费者洞察专家

**输入：**
- 品牌名称
- 产品品类
- 用户评论数据

**输出：**
- 用户画像（年龄、收入、地域、消费习惯）
- 核心需求与痛点
- 市场趋势与机会

**知识库：**
- 用户评论（电商平台、社交媒体）
- 母婴行业报告
- 市场调研数据

**工具：**
- RAG（向量检索）

**Prompt：**
```
你是一名资深消费者洞察专家，专注于母婴棉品行业。
请分析用户评论和行业数据，输出结构化的消费者洞察报告。
```

---

### 2. Product Manager Agent

**身份：**
产品经理

**输入：**
- 消费者洞察报告
- 品牌调性
- 竞品信息

**输出：**
- 产品名称
- 产品定位
- 目标用户
- 核心卖点
- 创新点

**知识库：**
- 竞品分析报告
- 产品规则与规范
- 品牌战略文档

**工具：**
- RAG（向量检索）

**Prompt：**
```
你是一名资深产品经理，专注于母婴棉品行业。
请根据消费者洞察设计新品方案，输出完整的产品定义。
```

---

### 3. Marketing Agent

**身份：**
营销运营专家

**输入：**
- 产品方案
- 品牌调性
- 目标用户

**输出：**
- 小红书文案（标题+正文+标签）
- 抖音脚本（分镜+台词）
- 电商详情页（标题+卖点+描述）
- 社群文案（种草+转化）

**知识库：**
- 品牌资料（品牌故事、视觉规范）
- 优秀营销案例
- 平台规则与玩法

**工具：**
- RAG（向量检索）
- Dify Workflow

**Prompt：**
```
你是一名资深营销运营专家，专注于母婴棉品行业。
请根据产品方案生成多平台营销内容，确保内容符合品牌调性。
```

---

### 4. Growth Agent

**身份：**
增长分析师

**输入：**
- 营销内容
- 投放数据（可选）
- 竞品表现（可选）

**输出：**
- 预期效果预测（点击率、转化率、预估收入）
- 优化建议（各渠道）
- 增长策略与行动计划

**知识库：**
- 历史营销数据
- 增长分析报告
- 行业基准数据

**工具：**
- RAG（向量检索）
- 数据分析

**Prompt：**
```
你是一名资深增长分析师，专注于母婴棉品行业。
请分析营销内容，预测效果并提出优化建议和增长策略。
```

---

## Data Flow

```
输入: brand, category
    │
    ▼
Insight Agent ──► 消费者洞察报告
    │
    ▼
Product Manager Agent ──► 产品方案
    │
    ▼
Marketing Agent ──► 营销内容
    │
    ▼
Growth Agent ──► 增长分析报告
```
